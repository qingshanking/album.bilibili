获取相册列表

接口方式

请求地址 版本号 请求方式 返回数据
https://v2.album.bilibili.xiaose.llili.top/getlist.php V2.0.200307 GET/POST json

请求参数

参数名称 必填 类型 说明
page 是 int 页码

返回数据

名称 类型 说明
code int 请求状态(0成功1失败)
msg string 失败原因
data object 成功数据，参考data返回值

data数据

名称 类型 说明
title string 相册标题
pictures object 图片数组
doc_id int 文章id
userhead string 作者头像
author string 作者名称

pictures数据

名称 类型 说明
img_src string 图片地址
img_width int 图片宽度
img_height int 图片高度
img_size int 图片大小
